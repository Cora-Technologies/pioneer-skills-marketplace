# pioneer-core

Essential Pioneer workflows for every employee. Covers the communication and
productivity tasks that come up every week.

## Skills

| Skill | Trigger phrases | What it does |
|-------|----------------|-------------|
| `meeting-notes` | "write up my meeting notes", "summarize this meeting" | Structures raw notes or transcripts into a clean, shareable summary with decisions and action items |
| `weekly-digest` | "weekly digest", "week in review", "what happened this week" | Compiles a weekly summary across projects — useful for personal review or sending to your team |
| `status-update` | "write a status update", "project update for [person]" | Drafts a concise stakeholder update with status (on track / at risk / blocked), progress, and next steps |

## Installation

Download `pioneer-core.plugin` from the [marketplace](../../README.md) and
install via **Settings → Plugins → Install from file** in Cowork.
